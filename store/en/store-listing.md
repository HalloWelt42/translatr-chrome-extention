## Detailed Description

```txt
Smart Translator translates websites directly in your browser -- fully local, no cloud services, no tracking.

TWO BACKENDS TO CHOOSE FROM

LibreTranslate
- Open-source translation engine, self-hosted or public instance
- Fast, no GPU needed, 30+ languages
- Perfect for everyday translations

LM Studio (Local LLM)
- Uses local AI models for high-quality translations
- Domain contexts: Automotive, Technical/IT, Medical, Legal
- Alternative translation suggestions
- Recommended: Qwen3-Next 80B-A3B (MLX) on Apple Silicon

FEATURES

- Full page translation -- one click, batch processing with progress indicator
- Selection translation -- select text, click icon, instant translation
- Side panel -- translation panel next to the webpage with advanced features
- Hover original -- hover over translated elements to see original text
- Text-to-speech -- read aloud via browser speech synthesis
- Markdown export -- export translated pages as Markdown
- Domain strategies -- optimized filters for Wikipedia, GitHub, StackOverflow etc.

TRANSLATION CACHE

- Browser cache (LocalStorage) -- store translations locally in the browser
- SWT Cache Server (Docker) -- central cache server for multiple devices
- Hybrid mode -- combine both for maximum speed
- Previously translated texts load instantly from cache

PRIVACY

- No cloud services -- all data stays local
- No tracking, no cookies, no telemetry
- No ads, no analytics
- All servers are self-hosted by the user

OPEN SOURCE

Smart Translator is open source under a non-commercial license.
Source code: https://github.com/HalloWelt42/smart-translator
```
