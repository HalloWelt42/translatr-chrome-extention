## Ausführliche Beschreibung

```txt
Smart Translator übersetzt Webseiten direkt im Browser -- komplett lokal, ohne Cloud-Dienste, ohne Tracking.

ZWEI BACKENDS ZUR WAHL

LibreTranslate
- Open-Source Übersetzungs-Engine, selbst gehostet oder öffentliche Instanz
- Schnell, keine GPU nötig, 30+ Sprachen
- Ideal für alltägliche Übersetzungen

LM Studio (Lokales LLM)
- Nutzt lokale KI-Modelle für hochwertige Übersetzungen
- Fachkontexte: Kfz/Automotive, Technisch/IT, Medizin, Recht
- Alternative Übersetzungsvorschläge
- Empfohlen: Qwen3-Next 80B-A3B (MLX) auf Apple Silicon

FUNKTIONEN

- Ganze Seiten übersetzen -- mit einem Klick, batch-weise mit Fortschrittsanzeige
- Textauswahl übersetzen -- Text markieren, Icon klicken, sofort übersetzen
- Side Panel -- Übersetzungs-Panel neben der Webseite mit erweiterten Funktionen
- Hover-Original -- Originaltext beim Überfahren übersetzter Elemente anzeigen
- Text-to-Speech -- Vorlesefunktion über Browser-Sprachsynthese
- Markdown-Export -- Übersetzte Seiten als Markdown exportieren
- Domain-Strategien -- Optimierte Filter für Wikipedia, GitHub, StackOverflow u.a.

ÜBERSETZUNGS-CACHE

- Browsercache (LocalStorage) -- Übersetzungen lokal im Browser speichern
- SWT Cache Server (Docker) -- Zentraler Cache-Server für mehrere Geräte
- Hybrid-Modus -- Beides kombiniert für maximale Geschwindigkeit
- Einmal übersetzte Texte werden sofort aus dem Cache geladen

DATENSCHUTZ

- Keine Cloud-Dienste -- alle Daten bleiben lokal
- Kein Tracking, keine Cookies, keine Telemetrie
- Keine Werbung, keine Analytics
- Alle Server werden vom Nutzer selbst betrieben

OPEN SOURCE

Smart Translator ist Open Source unter einer nicht-kommerziellen Lizenz.
Quellcode: https://github.com/HalloWelt42/smart-translator
```
